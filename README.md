# Shooting Game C++ (Work in Progress)

## Project Description

This is a 2D shooting game developed in C++.

The player controls a spaceship that can move horizontally and shoot projectiles using the keyboard.

Positions are represented using complex numbers.

Projectiles move vertically upward.

Enemies are placed at the top of the screen and are initially static.

The game checks collisions between projectiles and enemies.
Enemies are represented as rectangles, and projectiles as circles.
The project also uses complex numbers and particle-based movement concepts.
